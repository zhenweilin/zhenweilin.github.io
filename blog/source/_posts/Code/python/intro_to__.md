In the page, I will discuss five underscores' pattern and how they influence python program.
- single leading underscore: _var
- single end underscore: var_
- double leading underscore: __var
- double leading and end underscores: __var__
- single underscore: _

### Single and double underscores


Single and double underscores have some special meanings in python. Some meanings are conventions and are regarded as reminders to programmer, while some are strictly enforced.
